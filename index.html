Quién soy



Me llamo Nancy Josefina Páez Acosta, mi intención es ayudarte con una idea, empresa, servicio, marca u otro proyecto que estés impulsando.

Mi trabajo es la creación de contenido efectivo, profesional, coherente y original.

El servicio de creación de contenido que ofrezco es todo un proceso que combina la investigación, redacción, corrección, edición y otras técnicas para dar con resultado completo y perfecto. ¡Si me gusta creer que encuentro la perfección escribiendo!

Mi enfoque es entender el 100% el proyecto que llega a mis manos y conversar con el creador en busca de mayor claridad. Además, siempre estoy disponible a recibir retroalimentación y hacer cambios para lograr el contenido esperado.

Servicios



Solicita todo el contenido escrito que necesitas, ya sea para una marca personal o para una empresa.

Te puedo ayudar con los textos informativos para blogs, guías de compras, redacción de servicios, entre otros.

Los servicios incluyen: Investigación para la creación del texto. La redacción de un texto 100% original y libre de plagio. Corrección de gramática. Una vez entregado el pedido, tendrás hasta tres opciones de corrección si así lo deseas.
